# termo-py
Uma versão do famoso jogo TERMO feita em Python.
Projeto para estudo de interface gráfica e outras bibliotecas famosaas em Python.

- Parte gráfica feita com biblioteca Tkinter
- Integração de planilha CSV com a biblioteca Pandas para adição de palavras
- Conversão de objetos da planilha para array com Numpy

NOTA: O projeto ainda está em desenvolvimento, novas funcionalidades serão atualizadas posteriormente.
